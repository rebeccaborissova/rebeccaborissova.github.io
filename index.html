My Projects

Minesweeper

A recreation of the classic online game "Minesweeper" in C++ and SFML. This project required understanding how to use graphics libraries, performing in-game event handling, understanding the behavior of each tile, and structuring the game using encapsulation.

Blackjack Statistics

This is a GUI which allows you to simulate a game of blackjack using three different strategies: basic strategy, advanced strategy, and card counting. The program then calculates various statistics for all three strategies and displays them to the user. This project used python to create the blackjack simulation, pygame to create the GUI, as well as scipy and matplotlib to visualize the statistics.

Truck Insurance Scraper

Navigates through the US Department of Transportation website to collect data on truck insurance policies. The scraper uses the pyautogui and selenium packages with python to navigate the website, and saves the relevant data into a csv file.
