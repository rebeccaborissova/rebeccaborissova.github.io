Truck Insurance Scraper

This script navigates the US Department of Transportation website to scrape data about truck insurance policies. The program uses the selenium and pyautogui packages with python to automatically navigate the website. This program collects data on truck insurance companies, the date their truck insurance expires, and the type of truck insurance they have. This data is then cleaned and saved into a csv file to be easily parsed later.
Note: Script is run in GUI browser and slowed down for demonstration purposes. CSV file shown is from a previous run of the scraper.
